Dreamweaver 是什么

Dreamweaver（DW）是 Adobe 的网页编辑工具，既能可视化排版，也能写 HTML/CSS/JS 代码。

  • 适合：传统“切页面/企业站”工作流、快速拼页面。
  • 现代开发更常用：VS Code + 浏览器预览 + 构建工具（Vite/Next）。

三列式网页是什么

三列式 = 左侧栏（导航/目录）+ 中间主内容 + 右侧栏（推荐/工具/速览）。

  • 优点：信息分区清晰，文档站/资讯站常见。
  • 移动端：通常折叠侧栏，变单列或抽屉菜单。

Java 如何发送 HTML/CSS

你的 Java 小服务器本质只做一件事：把文件内容按 URL 返回给浏览器。

  1. 用户访问 / → 返回 index.html
  2. HTML 里引用 /style.css → 浏览器自动再请求一次 → 返回 style.css

关键点：

  • index.html 里必须写：<link rel="stylesheet" href="/style.css">
  • Java 端必须对 /style.css 做 createContext 并设置 Content-Type: text/css

现代网站通常怎么做

现代做法更强调：内容来源、组件化、路由、性能与维护成本。

  • 文档/教程类：Markdown/MDX + 自动 TOC + SSG（VitePress/Docusaurus/Next）
  • 业务系统：React/Vue + API + 组件库（AntD/Element）
  • 后端：更多提供 API；页面渲染交给前端或静态生成
